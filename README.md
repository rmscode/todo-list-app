
# Simple To-Do

A super simple to-do list app


## Authors

- [@RickySaull](https://www.github.com/rmscode)


## Features

- Add an item to a list
- Mark items as done
- Delete an item from the list
- Change the order of items in the list


## Tech Stack

**Client:** HTML, CSS, JavaScript, jQuery

**Server:** None
